Pourquoi 432 ?

La structure mathématique d'un temps vivant, fractal et spiralé.

432
La structure
432 = 12 × 36. Douze phases de maturation, chacune de 36 jours. Un cycle complet de soi.
12

Émergence (j.1–12) · Expansion (j.13–24) · Intégration (j.25–36). Chaque phase respire en trois temps.
36
Cycles par grand cycle
12 axes × 3 cycles chacun = 36 cycles = 15 552 jours. Un grand cycle complet.

Comment le Cycle se Calcule

Transparence complète sur les formules et les points clés.

J0 — Le Point d'Ancrage
Date de naissance ou de commencement. Tout le système se calibre depuis ce point.
J+288 — Le Pivot Fondamental
Exactement 288 jours après le début du cycle actuel. Fin de la phase 8 (Transformation). Le cycle bascule vers l'intégration.
// Depuis J0 jusqu'à aujourd'hui
N jours écoulés depuis J0

// Position dans le grand cycle
Grand cycle = floor(N ÷ 15552) // 36 × 432 = 15552 jours
Jour dans le grand cycle = N mod 15552

// Position dans le cycle actuel
Numéro du cycle = floor(jour_grand_cycle ÷ 432) // 0 à 35
Jour dans le cycle = jour_grand_cycle mod 432 // 0 à 431 → affich. 1 à 432

// Phase et jour dans la phase
Phase = floor(jour_cycle ÷ 36) // 0 à 11 → Phase 1 à 12
Jour dans la phase = (jour_cycle mod 36) + 1 // 1 à 36

// Zone spirale
Zone spirale = jour_cycle ≥ 360 // phases 11–12

// J+288 du cycle en cours
J0_cycle = J0 + (grand_cycle × 15552 + num_cycle × 432) jours
J+288 = J0_cycle + 288 jours
