# 🖼️ Fonds d'écran — 18 images

Dépose ici tes **18 fonds d'écran** au format **PNG**.

## Nommage (obligatoire)

Les fichiers doivent être nommés très exactement ainsi :

```
img/
  bg-01.png
  bg-02.png
  bg-03.png
  …
  bg-18.png
```

Dès qu'un fichier existe, il remplace **automatiquement** le dégradé de secours
correspondant — aucune manipulation, aucune config. Si un numéro manque, son
dégradé coloré reste affiché (l'écran ne casse jamais).

## Conseils

| Paramètre   | Valeur conseillée                          |
|-------------|--------------------------------------------|
| Format      | PNG (ou JPG renommé en .png)               |
| Orientation | Portrait (téléphone) — ex. 1080 × 1920 px  |
| Poids       | Viser < 400 Ko/image (APK léger, < ~7 Mo)  |

> Les images sont embarquées dans l'application → affichage instantané,
> **100 % hors-ligne**, aucune connexion nécessaire.

## Choisir un fond dans l'appli

Menu (☰) → **Fonds d'écran** → tape une vignette. Le choix est mémorisé.
